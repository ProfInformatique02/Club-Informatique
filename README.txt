ثانوية حساني محمد - موقع تجريبي لنادي الإعلام الآلي
========================

محتويات الحزمة:
- index.html
- activities.html
- contact.html
- assets/css/style.css
- assets/js/main.js
- assets/img/logo.svg

كيفية الاستخدام:
1. فك ضغط الملف ZIP (أو ضع الملفات في مجلد واحد).
2. افتح index.html في متصفح حديث (Chrome, Firefox).
3. استخدم زر 'Français' لتبديل اللغة بين العربية والفرنسية.
4. التصميم احترافي مع تأثيرات بسيطة ويمكن للتلاميذ تعديله بسهولة.

ملاحظات:
- النموذج لا يتضمن backend. نموذج الاتصال هو تجريبي فقط.
- لتحميل على GitHub Pages: انشر ملفات المجلد الجذري إلى repository واضبط GitHub Pages ليأخذ المحتوى من الفرع الرئيسي.
